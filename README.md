## Laravel Breeze Book Store

This is a Laravel Breeze-based book store project.

### Features
- Home screen
- Profile modal (using Breeze component)
- Book page to show book details
- Users can like books
- Repository pattern for data access
- Book details can be fetched from abc.nl (add logic as needed)

### Setup
1. Install dependencies:
   ```powershell
   composer install
   npm install
   ```
2. Run migrations:
   ```powershell
   php artisan migrate
   ```
3. Start the development server:
   ```powershell
   php artisan serve
   npm run dev
   ```

### Customization
- Add logic to fetch book details from abc.nl in `BookController@show`.
- Extend repository classes as needed.
